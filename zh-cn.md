### 协议、工具与编排框架

- [Model Context Protocol](https://modelcontextprotocol.io/introduction) - 连接模型、工具与外部上下文的开放协议
- [PydanticAI](https://ai.pydantic.dev/) - Pydantic 团队推出的类型安全 Python Agent 框架
- [Mastra 文档](https://mastra.ai/docs) - 带工作流、评测、记忆与可观测性的 Agent 框架
- [CrewAI 文档](https://docs.crewai.com/) - 多智能体角色编排框架官方文档
- [LangGraph](https://github.com/langchain-ai/langgraph) - 图结构、有状态 Agent 工作流框架
- [LangChain Agent 教程](https://python.langchain.com/docs/tutorials/) - LangChain 官方 Agent 与工具调用教程
- [Microsoft AutoGen](https://github.com/microsoft/autogen) - 多智能体对话框架
- [Dapr Python SDK](https://github.com/dapr/python-sdk) - 构建分布式与有状态 Python 服务的基础组件

### 参考项目与学习资料

- [CrewAI 示例](https://github.com/crewAIInc/crewAI-examples) - CrewAI 团队维护的示例项目

- [SuperModels](https://github.com/JohannesVC/SuperModels) - 带反思机制的桌面 Agent 运行时
- [AgentKit](https://github.com/japanvik/agentkit) - 面向网络化与分布式 Agent 的轻量工具集
- [OpenAI Cookbook](https://cookbook.openai.com/) - OpenAI 官方示例与生产实践集合
- [Prompt 优化指南](https://cookbook.openai.com/examples/optimize_prompts) - OpenAI 的提示词优化工作流
- [使用 CrewAI 学习 Agentic AI](https://github.com/panaversity/learn-agentic-ai) - 较完整的 Agentic AI 学习仓库
