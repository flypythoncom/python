## Agent Development

### Official SDKs and Platform Docs

- [OpenAI Agents Python Documentation](https://openai.github.io/openai-agents-python/) - Official Python SDK docs for building multi-step agents
- [OpenAI Agents SDK Quickstart](https://openai.github.io/openai-agents-python/quickstart/) - Official quickstart for getting a Python agent running fast
- [OpenAI Python Library](https://github.com/openai/openai-python) - Official Python client for the OpenAI API
- [OpenAI Developer Resources](https://developers.openai.com/resources) - Official resource hub for agents, tools, prompts, and starter apps
- [Anthropic Tool Use](https://docs.anthropic.com/en/docs/agents-and-tools/tool-use/overview) - Official guide to tool calling and agent workflows with Claude
- [Google Agent Development Kit](https://google.github.io/adk-docs/) - Official Google ADK docs for agent systems

### Protocols, Tooling, and Orchestration

- [Model Context Protocol](https://modelcontextprotocol.io/introduction) - Open protocol for connecting models to tools and external context
- [PydanticAI](https://ai.pydantic.dev/) - Type-safe Python agent framework from the Pydantic team
- [Mastra Docs](https://mastra.ai/docs) - Agent framework with workflows, evals, memory, and observability
- [CrewAI Documentation](https://docs.crewai.com/) - Official docs for role-based, multi-agent orchestration
- [LangGraph](https://github.com/langchain-ai/langgraph) - Stateful workflow framework for graph-based agents
- [LangChain Agent Tutorials](https://python.langchain.com/docs/tutorials/) - Official tutorials for tool use and agent patterns
- [Microsoft AutoGen](https://github.com/microsoft/autogen) - Multi-agent conversation framework
- [Dapr Python SDK](https://github.com/dapr/python-sdk) - Useful building blocks for stateful and distributed Python services

### Reference Projects and Learning Material

- [CrewAI Examples](https://github.com/crewAIInc/crewAI-examples) - Working CrewAI examples maintained by the CrewAI team
- [min-agent](https://github.com/zhouzaida/min-agent) - Minimal implementation for understanding core agent mechanics
- [OpenSource LLM Agents ReAct](https://github.com/Praveengovianalytics/llm_agents_open_source) - Open-source ReAct style implementation references
- [SuperModels](https://github.com/JohannesVC/SuperModels) - Desktop-focused agent runtime with reflection mechanisms
- [AgentKit](https://github.com/japanvik/agentkit) - Lightweight toolkit for networked and distributed agents
- [OpenAI Cookbook](https://cookbook.openai.com/) - Official examples and practical patterns for production systems
- [Prompt Optimization Guide](https://cookbook.openai.com/examples/optimize_prompts) - Practical prompt improvement workflow from OpenAI
- [Learn Agentic AI with CrewAI](https://github.com/panaversity/learn-agentic-ai) - Broad course repository for agentic AI experiments
